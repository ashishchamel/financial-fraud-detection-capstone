This dataset contains anonymized credit card transaction records used for fraud detection analysis.

- File: Financial_Fraud_Detection_Datasets.zip
- Size: ~45 MB (compressed)
- Records: 389,002
- Columns:
  cc_num, category, amt, gender, job, city_pop, trans_date_trans_time, is_fraud, etc.
- Target Variable: is_fraud (0 = genuine, 1 = fraudulent)
- Notes:
  - Data anonymized for educational use.
  - Original dataset stored offline due to size constraints.
